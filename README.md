dotfiles
========

repo for my dotfiles

## Assumptions
* **Homebrew** installed
* **iTerm2** installed

#### Homebrew installs
* **brew-cask** - install apps via brew - https://github.com/phinze/homebrew-cask
* **zsh**
* **vim**
* **rbenv** **ruby-build**

#### Submodules
* **zprezto** - configuration framework for Zsh - https://github.com/sorin-ionescu/prezto
